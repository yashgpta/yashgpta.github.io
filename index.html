Optimization of FFT Based Image Compression

Intelligent Water Drop Algorithm(IWD)

Applied Intelligent Water Drop Algorithm (IWD) to optimize Fast Fourier Transform (FFT) based image compression. Achieved optimized threshold FFT compression parameters using the IWD Algorithm & used Structural Similarity index (SSIM) to show comparison between the traditional FFT based compression and IWD based FFT compression. Primary author of the technical paper, which is under review as a Journal paper at SN Applied Sciences.

Client-Server Cloud Computing Analysis
